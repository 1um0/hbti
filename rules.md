# HBTI 徒步人格测试 V2.0 完整版
（Hiking Behavioral Type Indicator｜徒步圈专属SBTI风人格测试）
## 一、测试定位
基于户外行为心理学、群体社交习惯原创，**36题5分量表**，测出你的4字母徒步人格，**文案戏谑扎心、徒步圈黑话拉满**，不正经但巨准。

## 二、四大核心维度（二元对立）
1. **能量维度**：S 内倾独行 ↔ G 外倾结伴（独处回血 / 社交快乐）
2. **行动维度**：P 严谨规划 ↔ F 随性自由（攻略拉满 / 说走就走）
3. **动机维度**：C 征服挑战 ↔ B 沉浸共生（冲顶内卷 / 躺平赏景）
4. **耐受维度**：H 硬核极限 ↔ S 舒适轻享（越虐越爽 / 舒服至上）

---

## 三、测试规则（5分钟做完，零门槛）
1. 共36题，**1-5分自评**：1=完全不符合，2=不太符合，3=中立，4=比较符合，5=完全符合
2. 标注**【反向】**的题目需反转计分：5↔1、4↔2、3不变
3. 题目分4组，每组9题，单独计算平均分，判定维度倾向
4. 判定标准：**均分≥3.5=前字母；均分＜3.5=后字母**
5. 四字母按顺序组合，即为你的最终HBTI徒步人格

## 五、标准化评分体系
1. 计分步骤：处理反向题 → 每组9题求和 → 计算**平均分=总分÷9**
2. 维度判定：
   - 能量：≥3.5=S 独行；＜3.5=G 结伴
   - 行动：≥3.5=P 规划；＜3.5=F 随性
   - 动机：≥3.5=C 征服；＜3.5=B 沉浸
   - 耐受：≥3.5=H 硬核；＜3.5=S 轻享
3. 结果生成：**S/G + P/F + C/B + H/S** 四字母组合
