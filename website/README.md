# HBTI徒步人格测试网站

![HBTI Logo](https://img.shields.io/badge/HBTI-徒步人格测试-blue)

基于户外行为心理学的原创徒步人格测试系统，通过36道5分量表题测出你的4字母徒步人格，提供戏谑扎心的人格解析与徒步搭子匹配建议。

## ✨ 项目特色

- **原创心理学模型**：基于户外行为心理学、群体社交习惯原创的4维度人格模型
- **完整测试体系**：36道精心设计的测试题，涵盖徒步场景的各个维度
- **戏谑人格解析**：16种人格的毒舌扎心描述，徒步圈黑话拉满，不正经但巨准
- **纯前端实现**：无需后端服务器，所有计算在本地完成，保护用户隐私
- **响应式设计**：适配手机、平板、电脑等多种设备
- **一键部署**：支持GitHub Pages免费部署，零成本上线

## 📋 测试维度

HBTI测试基于四大核心维度，每个维度对应一个字母组合：

| 维度 | 代码 | 前字母倾向 | 后字母倾向 | 描述 |
|------|------|------------|------------|------|
| 能量维度 | S/G | S 内倾独行 | G 外倾结伴 | 评估徒步中的社交偏好与能量来源 |
| 行动维度 | P/F | P 严谨规划 | F 随性自由 | 评估徒步规划习惯与行动风格 |
| 动机维度 | C/B | C 征服挑战 | B 沉浸共生 | 评估与自然的互动方式与徒步动机 |
| 耐受维度 | H/S | H 硬核极限 | S 舒适轻享 | 评估徒步强度偏好与耐受能力 |

## 📝 测试规则

### 计分规则
1. **题目数量**：共36题，分为4个维度，每个维度9题
2. **评分范围**：1-5分（1=完全不符合，5=完全符合）
3. **反向题处理**：标注"反向"的题目需反转计分（5↔1、4↔2、3不变）
4. **维度计算**：每个维度9题总分 ÷ 9 = 维度平均分
5. **人格判定**：平均分 ≥ 3.5 = 前字母；平均分 < 3.5 = 后字母

### 人格生成
四个维度的判定字母按顺序组合，形成4字母人格代码：
```
能量维度 + 行动维度 + 动机维度 + 耐受维度 = 人格代码
示例：S (独行) + P (规划) + C (征服) + H (硬核) = SPCH
```

## 🔧 开发指南

### 修改测试题目
编辑 `app.js` 中的 `QUESTIONS` 数组，每个题目对象包含：
```javascript
{
  id: 1,                         // 题目编号
  text: '题目内容...',           // 题目文本
  dimension: 0,                  // 维度索引 (0-3)
  reverse: false                 // 是否为反向题
}
```

### 添加人格类型
编辑 `app.js` 中的 `PERSONALITY_TYPES` 对象：
```javascript
'SPCH': {
  name: '人格名称',
  nickname: '外号描述',
  description: '详细的人格描述...',
  group: '所属组别',
  rarity: '稀有度百分比',
  compatibility: {
    best: ['最佳搭子1', '最佳搭子2'],
    avoid: ['避雷组合1', '避雷组合2']
  }
}
```

### 自定义样式
- 修改 `index.html`、`test.html`、`result.html` 中的 `<style>` 标签
- 调整Tailwind CSS类名或添加自定义CSS

## 📊 数据与隐私

### 数据处理原则
- **本地计算**：所有计分逻辑在用户浏览器中完成
- **无数据上传**：测试答案和结果不会发送到任何服务器
- **临时存储**：使用localStorage暂存结果，刷新页面后失效
- **隐私保护**：不收集任何用户身份信息或敏感数据

### 数据格式
测试结果以JSON格式存储在本地：
```json
{
  "code": "SPCH",
  "averages": [4.2, 3.8, 4.5, 4.1],
  "scores": [38, 34, 41, 37],
  "letters": ["S", "P", "C", "H"]
}
```
