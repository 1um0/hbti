# HBTI徒步人格测试

基于户外行为心理学的原创徒步人格测试系统。通过36道5分量表题测出你的4字母徒步人格，了解自己的徒步特质，找到最匹配的登山搭子。

---

## ✨ 项目特色

- **原创心理学模型**：基于户外行为心理学、群体社交习惯原创的4维度人格模型
- **完整测试体系**：36道精心设计的测试题，涵盖徒步场景的各个维度
- **扎心人格解析**：16种人格的毒舌扎心描述，徒步圈黑话拉满，不正经但巨准
- **智能搭子匹配**：基于人格特质和动机差异，精准预测搭子兼容性
- **原创建议库**：每个人格享有专属的徒步建议与风险规避指南
- **纯前端设计**：无需后端，所有计算在本地完成，保护用户隐私

## 📋 测试维度

HBTI测试基于四大核心维度，揭示你在山野中的真实样子：

| 维度 | 代码 | 倾向A | 倾向B | 核心问题 |
|------|------|-------|-------|---------|
| **能量维度** | S/G | S 内倾独行 | G 外倾结伴 | 徒步时你的社交偏好是什么？ |
| **行动维度** | P/F | P 严谨规划 | F 随性自由 | 你的行动风格是精确规划还是随遇而安？ |
| **动机维度** | C/B | C 征服挑战 | B 沉浸共生 | 你想征服山峰还是感受山野？ |
| **耐受维度** | H/S | H 硬核极限 | S 舒适轻享 | 你追求极限虐线还是舒适体验？ |

**计算逻辑**：每个维度9道题，平均分≥3.5判定为前字母，<3.5判定为后字母，最终组合形成4字母人格代码。

## 📊 记分规则

### 1. 基础计分

| 评分等级 | 含义 |
|---------|------|
| 5分 | 完全符合 |
| 4分 | 基本符合 |
| 3分 | 不确定 |
| 2分 | 基本不符 |
| 1分 | 完全不符 |

### 2. 反向题处理

部分题目为反向表述（标注"反向"），计分需反转：
- 反向题评分5分 → 计算为1分
- 反向题评分4分 → 计算为2分  
- 反向题评分3分 → 保持3分
- 反向题评分2分 → 计算为4分
- 反向题评分1分 → 计算为5分

### 3. 维度平均分计算

每个维度9道题，计算公式：
```
维度平均分 = （所有题目得分之和）÷ 9
```

### 4. 人格字母判定

基于维度平均分与阈值（3.5）的比较：
- 维度平均分 **≥ 3.5** → 判定为该维度的**前字母**
- 维度平均分 **< 3.5** → 判定为该维度的**后字母**

### 5. 最终人格代码

按顺序组合四个维度的判定字母：
```
人格代码 = 能量维度字母 + 行动维度字母 + 动机维度字母 + 耐受维度字母

示例：
- 能量维度：3.8 ≥ 3.5 → S
- 行动维度：3.2 < 3.5 → F  
- 动机维度：4.1 ≥ 3.5 → C
- 耐受维度：3.6 ≥ 3.5 → H
- 最终人格 = SFCH（随性孤勇者）
```

## 🧭 搭子匹配原则

### 1. 山野匹配度算法

系统基于三个核心因素计算每个人的"山野匹配度"（50-99%）：

**① 基础准备度** (权重50%)
- 综合四个维度的平均分
- 反映整体徒步能力与自我认知

**② 目标清晰度** (权重30%)
- 衡量维度判定的明确程度
- 偏离平衡点（3.5）越远，清晰度越高
- 帮助预测行动的一致性

**③ 维度均衡度** (权重20%)
- 四个维度得分分布的均匀程度
- 均衡度高 → 适应性强、容错率高
- 均衡度低 → 特征明显、风格突出
